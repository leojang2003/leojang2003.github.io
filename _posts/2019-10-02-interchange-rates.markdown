### 關於 Interregional Interchange Rates

<br />

> These rates typically apply to transactions that are issued and acquired in different regions.

此費率用於跨區域的交易

<br />

#### Mastercard Interregional Consumer Rates

> Transactions meeting interchange program criteria qualify for associated rates.
滿足下列 interchange program 的交易適用以下費率

<br />

**Consumer POS Rates**

| IRD | Program Name        | Rate (USD)           |
| ------------- |:-------------:|:-------------:|
| IRD 20|Interregional Consumer Payment Transaction      | 0.19% + 0.53 |
| IRD 24|Interregional Consumer Merchant UCAF     | 1.44% + 0.00      |
| IRD 2A|Interregional Mastercard Initiated Rewards |0.00% + 0.00      |
| IRD 73, 83|Interregional Consumer Electronic |1.10% + 0.00      |
| IRD 75, 85, 95|Interregional Consumer Standard |1.60% + 0.00      |
| IRD 79|Interregional Consumer Full UCAF |1.54% + 0.00      |
| IRD MS|Interregional Mastercard MoneySend Payment |0.00% + 0.00      |

<br />
